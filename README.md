# ALFAHISS Project

## [English]
**ALFAHISS** is a responsive, bilingual (English/Swedish) website developed for an elevator maintenance company. The application is built using **Django** and utilizes **Bootstrap** for front-end design.

---

## [Ελληνικά]
Το **ALFAHISS** είναι μια responsive, δίγλωσση (Αγγλικά/Σουηδικά) ιστοσελίδα που αναπτύχθηκε για τις ανάγκες εταιρείας συντήρησης ανελκυστήρων, με χρήση **Django** και **Bootstrap**.

---

## Technologies / Τεχνολογίες
* **Backend:** Python, Django 5.2.14
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **I18n:** Django Internationalization (English and Swedish support)
* **Deployment:** Render

---

## Key Features / Κύρια Χαρακτηριστικά
* **Bilingual Interface:** Seamless language switching (English/Swedish).
* **Responsive Design:** Fully adaptive layout for mobile and desktop.
* **Dynamic Carousel:** Interactive homepage slider for services.
* **Admin Panel:** Easy content management via Django Admin.

## Installation & Setup / Εγκατάσταση και Εκτέλεση

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/your-username/ALFAHISS_project.git](https://github.com/your-username/ALFAHISS_project.git)
   cd ALFAHISS_project
2. **Create a Virtual Environment:** ```python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate```
3. **Install dependencies:** ```pip install -r requirements.txt```
4. **Run Migrations:** ```python manage.py migrate```
5. **Run Server:** ```python manage.py runserver```
